Configuration Supabase

Connecter votre instance Supabase pour activer la vraie base de données, l'authentification sécurisée et le Row Level Security.

⚠️ Important : La Service Role Key ne doit jamais être incluse dans du code client (JavaScript exposé dans le navigateur). Elle est réservée aux fonctions serveur (Edge Functions Supabase).
